# Banking System Manual Testing Project

QA manual testing project covering test case design, execution, bug reporting, and retesting for a banking application.

## Covered Features

### Manager
- New Customer
- Edit Customer
- Delete Customer
- New Account
- Edit Account
- Delete Account
- Mini Statement
- Customized Statement
- Fund Transfer
- Deposit
- Withdrawal
- Change Password
- Balance Enquiry

### Customer
- Change Password
- Balance Enquiry
- Mini Statement
- Customized Statement
- Fund Transfer

## Testing Activities
- Test Case Design
- Test Execution
- Defect Reporting
- Retesting (V1, V2, V3, V4)

## Deliverables
- Test_Cases.xlsx
- Bug_Report.xlsx

## Defects Identified
Examples include:
- HTTP 500 errors
- Validation issues
- Incorrect error messages
- Field inconsistencies
